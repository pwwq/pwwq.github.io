Debian安装LXDE VNC桌面和Firefox浏览器

2015-05-23
教程
次阅读

LXDE桌面环境相对其它VNC更加节省资源，开始安装

apt-get update
apt-get install xorg lxde-core tightvncserver -y

启动VNC/输入设置密码

tightvncserver :1

暂停VNC

tightvncserver -kill :1

配置xstartup系统文件

nano ~/.vnc/xstartup

然后在文件的最后添加下面的脚本代码

lxterminal &
/usr/bin/lxsession -s LXDE &

客户端下载 Download VNC Viewer 使用ip:5901登录

如果vnc远程桌面 桌面空白,编辑文件~/.vnc/xstartup将里面的最后一行twn &改成gnome-session & (即可视化界面)

安装Firefox浏览器

apt-get install iceweasel -y

Firefox浏览器文件下载路径在/tmp/mozilla_root0

可以使用free -m查看内存使用情况

LXDE桌面VNC和Firefox浏览器，安装完成后占用磁盘大概1GB左右。
