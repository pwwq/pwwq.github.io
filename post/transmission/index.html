安装Transmission#

apt install transmission-daemon

配置Transmission#

先停止Transmission

systemctl stop transmission-daemon.service

然后编辑配置文件/var/lib/transmission-daemon/info/settings.json

nano /var/lib/transmission-daemon/info/settings.json

WebUI美化#

安装transmission-web-control进行美化

chmod +x install-tr-control-cn.sh
bash install-tr-control-cn.sh

按照提示，输入相应的数字，按回车即可

管理Transmission#

service transmission-daemon stop
service transmission-daemon start

通过浏览器访问 http://IP:9091 WEBUI管理使用

制作种子#

find / -name transmission-create

寻找transmission安装目录

transmission-create

获得参数说明帮助

Options:
 -h --help                    Display this help page and exit
 -p --private                 Allow this torrent to only be used with the
                              specified tracker(s)
 -o --outfile   <file>        Save the generated .torrent to this filename
 -s --piecesize <size in KiB> Set how many KiB each piece should be, overriding
                              the preferred default
 -c --comment   <comment>     Add a comment
 -t --tracker   <url>         Add a tracker's announce URL
 -V --version                 Show version number and exit

参数说明：

  1. -p 表示这是私用的种子，这个 PT 必须要加上
  2. -o 生成的种子输出位置，不要忘记把名字打上
  3. -t tracker 服务器地址，自行查询各大站点
  4. -s 每个文件块的大小，单位是 KB，设置的 4096，也就是 4M
  5. 最后空一格写源文件的位置，也就是文件的存放位置，可以是一个文件或者一整个目录

示范

transmission-create -p -o /种子输出目录/种子文件名.torrent -t https://pt.m-team.cc/announce.php -s 4096 /要做种的文件目录/文件名

文件的命名里不要出现空格，会生成错误的文件

发布种子#

  1. 将制作好的种子文件.torrent发布到相关PT站点
  2. 请不要将视频移动、改名、删除，可以让更多人下载到这个种子。
  3. 上传种子后一定要下载网站的新种子文件，然后选择资源的上级目录，开始做种。
  4. 网站会统计每一个用户的下载量和上传量，下载量和上传量在一定程度上决定着用户的等级，有没有权限下载文件。每一个用户注册后会得到一个passkey，用户从网站里面下载的种子里面包含了私人的passkey。通过passkey识别每一个用户，统计每一个用户的下载、上传和做种时间等等。
