V2ray服务端的搭建配置

bash <(curl -L https://raw.githubusercontent.com/XTLS/Xray-install/main/install-release.sh)

免流的方案：VMess协议 + TCP + HTTP，编辑配置文件/usr/local/etc/xray/config.json，

nano /usr/local/etc/xray/config.json

完整配置示范如下

{
    "inbounds": [
        {
            "port": 80,
            "protocol": "vmess",
            "settings": {
                "clients": [
                    {
                        "id": "这里填写你的UUID"
                    }
                ]
            },
            "streamSettings": {
                "network": "tcp",
                "tcpSettings": {
                    "header": {
                        "type": "http",
                        "response": {
                            "version": "1.1",
                            "status": "200",
                            "reason": "OK",
                            "headers": {
                                "Content-Type": ["application/octet-stream", "video/mpeg"],
                                "Transfer-Encoding": ["chunked"],
                                "Connection": ["keep-alive"],
                                "Pragma": "no-cache"
                            }
                        }
                    }
                }
            }
        }
    ],
    "outbounds": [
        {
            "protocol": "freedom"
        }
    ]
}

可以通过命令cat /proc/sys/kernel/random/uuid随机生成

管理启动V2ray

systemctl enable xray
systemctl start xray
systemctl restart xray

v2ray 免流客户端配置

  • 别名(remarks)：随便填写
  • 地址(address): 服务器的IP地址或域名
  • 端口(port): 80 免流端口一般是80/8080/443，其它端口未测试
  • 用户ID: 填写服务器配置文件里的id
  • 额外ID：0
  • 加密方式(security): aes-128-gcm
  • 传输协议(network)：tcp
  • 伪装类型(type)：http
  • 伪装域名(host)(host/ws host/h2 host)/QUIC加密方式 tms.dingtalk.com
  • 底层传输安全(tls)：选择空
  • SNI:选择空
  • 跳过证书验证(allowlnsecure): 选择空

实现热点免流

安卓手机使用V2rayNG免root实现热点免流

打开V2rayNG依次进入【设置】【进阶设置】勾选【允许来自局域网的连接】记住下面的HTTP代理商品默认是10809在其它连接的客户端WIFI里面设置系统代理为手机的内网IP与v2ray的HTTP代理商品

可以解锁的安卓手机可以刷Magisk，然后装模块Xray4Magisk
