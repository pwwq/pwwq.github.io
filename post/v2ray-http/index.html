联通电信移动V2ray免流host http教程

2022年9月9日
| 教程

V2ray服务端的搭建配置

bash <(curl -L https://raw.githubusercontent.com/XTLS/Xray-install/main/install-release.sh)

免流的方案采用VMess协议+TCP+HTTP，编辑配置文件/usr/local/etc/xray/config.json，

nano /usr/local/etc/xray/config.json

完整配置示范如下

{
    "inbounds": [
        {
            "port": 80,
            "protocol": "vmess",
            "settings": {
                "clients": [
                    {
                        "id": "这里填写你的UUID"
                    }
                ]
            },
            "streamSettings": {
                "network": "tcp",
                "tcpSettings": {
                    "header": {
                        "type": "http",
                        "response": {
                            "version": "1.1",
                            "status": "200",
                            "reason": "OK",
                            "headers": {
                                "Content-Type": ["application/octet-stream", "video/mpeg"],
                                "Transfer-Encoding": ["chunked"],
                                "Connection": ["keep-alive"],
                                "Pragma": "no-cache"
                            }
                        }
                    }
                }
            }
        }
    ],
    "outbounds": [
        {
            "protocol": "freedom"
        }
    ]
}

可以通过命令cat /proc/sys/kernel/random/uuid随机生成

ws配置如下

管理启动V2ray

systemctl enable xray
systemctl start xray
systemctl restart xray

v2ray 免流客户端配置

  • 别名(remarks)：随便填写
  • 地址(address): 服务器的IP地址或域名
  • 端口(port): 80 免流端口一般是80/8080/443，其它端口未测试
  • 用户ID: 填写服务器配置文件里的id
  • 额外ID：0
  • 加密方式(security): aes-128-gcm
  • 传输协议(network)：tcp
  • 伪装类型(type)：http
  • 伪装域名(host)(host/ws host/h2 host)/QUIC加密方式 tms.dingtalk.com
  • 底层传输安全(tls)：选择空
  • SNI:选择空
  • 跳过证书验证(allowlnsecure): 选择空

实现热点免流

安卓手机使用V2rayNG免root实现热点免流

打开V2rayNG依次进入【设置】【进阶设置】勾选【允许来自局域网的连接】记住下面的HTTP代理商品默认是10809在其它连接的客户端WIFI里面设置系统代理为手机的内网IP与v2ray的HTTP代理端口

可以解锁的安卓手机可以刷Magisk，然后装模块Xray4Magisk下载Xray-android-arm64-v8a.zip安装Xray4Magisk-1.4.3.zip

编辑配置文件/data/adb/xray/confs/proxy.json替换成以下的

{
  "outbounds": [
    {
      "tag": "proxy",
      "protocol": "vmess",
      "settings": {
        "vnext": [
          {
            "address": "服务器的IP地址或域名",
            "port": 80,
            "users": [
              {
                "id": "改成你的ID参数",
                "alterId": 0,
                "security": "aes-128-gcm"
              }
            ]
          }
        ]
      },
      "streamSettings": {
        "network": "tcp",
        "tcpSettings": {
          "header": {
            "type": "http",
            "request": {
              "version": "1.1",
              "method": "GET",
              "path": [
                "/"
              ],
              "headers": {
                "Host": [
                  "tms.dingtalk.com"
                ],
                "User-Agent": [
                  "Mozilla/5.0 (Windows NT 10.0; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/55.0.2883.75 Safari/537.36",
                  "Mozilla/5.0 (iPhone; CPU iPhone OS 10_0_2 like Mac OS X) AppleWebKit/601.1 (KHTML, like Gecko) CriOS/53.0.2785.109 Mobile/14A456 Safari/601.1.46"
                ],
                "Accept-Encoding": [
                  "gzip, deflate"
                ],
                "Connection": [
                  "keep-alive"
                ],
                "Pragma": "no-cache"
              }
            }
          }
        }
      },
      "mux": {
        "enabled": false,
        "concurrency": -1
      }
    }
  ]
}

修成路由为全局代理，改配置文件/data/adb/xray/confs/routing.json替换成以下的

{
  "routing": {
    "domainStrategy": "IPIfNonMatch",
    "rules": []
  }
}
