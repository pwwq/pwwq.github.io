bitwarden_rs是一个免费开源、能自己搭建服务端的安全密码库服务器

  • https://bitwarden.com/
  • https://github.com/dani-garcia/bitwarden_rs
  • https://hub.docker.com/r/bitwardenrs/server

Docker安装

wget -qO- get.docker.com | bash

安装Docker Compose

curl -L https://github.com/docker/compose/releases/download/1.16.1/docker-compose-`uname -s`-`uname -m` > ./docker-compose
mv ./docker-compose /usr/bin/docker-compose
chmod +x /usr/bin/docker-compose

Docker进程管理

systemctl enable docker
systemctl start docker
systemctl status docker

部署 Bitwarden_rs

https://github.com/dani-garcia/bitwarden_rs

快速部署

docker pull bitwardenrs/server:latest
docker run -d --name bitwarden -v /bw-data/:/data/ -p 8800:80 bitwardenrs/server:latest

官方提供

curl -Lso bitwarden.sh https://go.btwrdn.co/bw-sh \
  && chmod +x bitwarden.sh
./bitwarden.sh install
./bitwarden.sh start

自定义部署(推荐使用)

拉取 bitwarden_rs 镜像

docker pull bitwardenrs/server:latest

生成一个admin用户管理页面的token(可选)

openssl rand -base64 48

运行 bitwarden_rs 容器

docker run -d --name bitwarden --restart=always \
    -e SIGNUPS_ALLOWED=true \
    -e INVITATIONS_ALLOWED=true \
    -e ADMIN_TOKEN=youpassword \
    -p 8800:80 \
    -v /bw-data/:/data/ \
    bitwardenrs/server:latest

参数解释：

  1. SIGNUP_ALLOWED是否允许注册
  2. INVITATIONS_ALLOWED是否允许组织邀请注册
  3. ADMIN_TOKEN用户管理界面 (/admin)，可用于删除用户及邀请用户注册
  4. 将宿主机的8800映射到容器的80端口上

更多参数可参考官方 Wiki

自己注册完成后建议将SIGNUP_ALLOWED设置为false禁止注册

容器操作命令

启动容器

docker start bitwarden

停止容器

docker stop bitwarden

删除容器

docker rm bitwarden

查看运行容器

docker ps -as

升级 Bitwarden 镜像

# 重新拉取镜像
docker pull bitwardenrs/server:latest
# 停止、删除原容器
docker stop bitwarden
docker rm bitwarden
# 重新运行 docker run 命令
# 查看镜像文件
docker image ls
# 删除原镜像文件, $ID 在 step4 中可以看到
docker image rm $ID

备份Bitwarden_rs

只需要备份/bw-data/db.sqlite3这个文件就行了。你可以用rclone sync同步到各个网盘(可选)

恢复Bitwarden_rs

先按上述方法安装运行Bitwarden_rs，然后停止

docker stop bitwarden

替换备份好的/bw-data/db.sqlite3这个文件即可，然后启动Bitwarden_rs

docker start bitwarden

如果你的数据库路径与原来的不一致，可以手动指定，代码示例：

docker run -d --name bitwarden \
  -e DATABASE_URL=/database/bitwarden.sqlite3 \
  -v /bw-data/:/data/ \
  -v /bw-database/:/database/ \
  -p 8800:80 \
  bitwardenrs/server:latest

bitwarden可以支持多用户注册使用，如果想要让bitwarden发送邮件的话，可以使用SMTP发邮件，参考代码如下：

docker run -d --name bitwarden \
  -e SMTP_HOST=<smtp.domain.tld> \
  -e SMTP_FROM=<bitwarden@domain.tld> \
  -e SMTP_PORT=587 \
  -e SMTP_SSL=true \
  -e SMTP_USERNAME=<username> \
  -e SMTP_PASSWORD=<password> \
  -v /bw-data/:/data/ \
  -p 8800:80 \
  bitwardenrs/server:latest

然后用WEB服务器(nginx,caddy等)弄个域名反代一下自定义端口,需要设置好SSL证书以https访问
