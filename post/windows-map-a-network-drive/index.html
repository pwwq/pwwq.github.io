Windows中将OneDrive映射为本地磁盘

2017-04-20
Windows 教程
次阅读

将 OneDrive 映射为本地网络驱动器 登录到 OneDrive 在浏览器地址栏即可获取账户 ID。类似：

https://onedrive.live.com/?id=root&cid=xxxxxxxxxxx

其中xxxxxxxxxxx为Onedrive账户ID

打开此电脑 计算机 映射网络驱动器

映射网络驱动器在文件夹输入

https://d.docs.live.net/xxxxxxxxxxx

按提示输入账号密码即能完成映射网络驱动器
